FOB Grid Format Instructions

Your FOB Grid Excel file should have the following format:

  • Required columns: Color, Sizes, FOB
  • The Sizes column can contain multiple sizes separated by spaces, slashes (/), backslashes (\), or commas
  • The tool will automatically split these sizes and create individual entries for each
  • FOB values should be numeric (e.g., 4.53, 5.05)
